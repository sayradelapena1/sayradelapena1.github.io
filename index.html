My Projects

I have created this website with the purpose of showcasing some of my projects in Excel, SQL, Tableau, and PowerBI. These projects demonstrate my proficiency in data management, analysis, and visualization across various platforms. From cleaning raw Excel files for housing projects using SQL queries to crafting insightful dashboards in Tableau and PowerBI, each project exemplifies my ability to extract actionable insights from data and present them effectively. I aim to highlight the value of data-driven decision-making and contribute to the advancement of data analytics practices.
